Скрипт загружает GTFS feed с сайта Организатора Перевозок, подсчитывает количество рейсов каждого маршрута, выполняющихся в будний день (понедельник) и выводит в файл (только "социальные" маршруты, т.к. для "коммерческих" данные в базе не соответствуют действительности).

Скрипт написан для быстрого и удобного сохранения данных для последующего анализа динамики изменения объёмов перевозок.

Для работы загрузчика файлов требуется модуль requests для Python.
