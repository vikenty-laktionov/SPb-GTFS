GTFS_v3.py: скрипт загружает GTFS feed с сайта Организатора Перевозок, подсчитывает количество рейсов каждого маршрута для типичного буднего и выходного дня (понедельник, суббота) и выводит в CSV-файл для дальнейшего анализа.

Скрипт написан для быстрого и удобного сохранения данных для последующего анализа динамики изменения объёмов перевозок.

Для работы загрузчика файлов требуются модули requests и pandas для Python и сертификаты Минцифры в папке со скриптом.

GTFS_v99.py: расширенная версия скрипта, вычисляюшая дополнительные параметры вроде интервалов по времени суток, средней скорости, километража за день, а так же добавляющая информацию из внешних источников

Требует внешний источник, базу данных по маршрутам routedata.csv
